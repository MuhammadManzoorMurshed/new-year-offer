# New Year Offer

A lightweight, static New Year promotional landing page built with plain HTML and CSS.

## Features
- Single-file HTML layout: `index.html` with styles in `css/new-year-offer.css`.
- Image and icon assets in `new-year-offer-resources/`.
- No build tools required

## Quick start
1. Open `index.html` in a browser to preview.
2. Edit text, images, or styles under `css/new-year-offer.css` to customize.
